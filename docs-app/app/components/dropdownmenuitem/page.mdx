## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `disabled` | `boolean` | No | false | Whether the item is disabled |
| `selected` | `boolean` | No | false | Whether the item is selected |
| `onSelect` | `(event: Event) => void` | No | - | Callback when item is selected
    Receives native Event (not SyntheticEvent) |
| `inset` | `boolean` | No | false | Whether item has inset padding (for nested items) |
