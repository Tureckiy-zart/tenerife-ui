# PopoverContent

Popover Content Component
 
  Content container for popover with positioning, animations, and Surface integration.
  Uses POPOVER_TOKENS for styling and positioning helper for placement.

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `placement` | `Placement` | No | bottom | Placement preference |
| `offset` | `number` | No | 4 | Offset between trigger and content (in pixels) |
| `closeOnInteractOutside` | `boolean` | No | true | Whether to close on outside click |
