## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `open` | `boolean` | Yes | - | Whether modal is open |
| `onClose` | `() => void` | Yes | - | Callback when modal should close |
| `size` | `"sm" | "md" | "lg" | "fullscreen"` | No | md | Modal size variant |
| `backdropVariant` | `"default" | "blurred" | "transparent"` | No | default | Backdrop variant |
| `closeOnBackdropClick` | `boolean` | No | true | Whether to close on backdrop click |
| `closeOnEscape` | `boolean` | No | true | Whether to close on escape key |
| `returnFocusRef` | `React.RefObject<HTMLElement>` | No | - | Element to return focus to when modal closes |
