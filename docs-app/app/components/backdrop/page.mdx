## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `variant` | `"default" | "blurred" | "transparent"` | No | default | Backdrop variant |
| `isVisible` | `boolean` | No | true | Whether backdrop is visible (for animation) |
| `onClick` | `(event: React.MouseEvent<HTMLElement>) => void` | No | - | Click handler for backdrop dismiss |
