## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `cols` | `ResponsiveColumns` | No | - | Number of columns (1-6, 12, or none) |
| `rows` | `ResponsiveRows` | No | - | Number of rows (1-6 or none) |
| `gap` | `ResponsiveSpacing` | No | - | Gap between grid items - token-based (xs, sm, md, lg, xl, 2xl, 3xl, 4xl, 5xl) |
| `flow` | `ResponsiveFlow` | No | - | Grid flow direction |
| `align` | `ResponsiveAlignment` | No | - | Align items |
| `justify` | `ResponsiveJustify` | No | - | Justify content |
