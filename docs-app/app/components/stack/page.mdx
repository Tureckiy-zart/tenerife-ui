## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `gap` | `ResponsiveSpacing` | No | - | Gap between stack items - token-based (xs, sm, md, lg, xl, 2xl, 3xl, 4xl, 5xl) |

| `justify` | `"start" | "end" | "center" | "between" | "around" | "evenly"` | No | - | Justify content |
