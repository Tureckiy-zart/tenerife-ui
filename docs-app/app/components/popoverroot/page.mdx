# PopoverRoot

Popover Root Component
 
  Root component for Popover that manages open/close state.
  Supports both controlled and uncontrolled modes.

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `open` | `boolean` | No | - | Whether the popover is open (controlled mode) |
| `onOpenChange` | `(open: boolean) => void` | No | - | Callback when open state changes |
| `defaultOpen` | `boolean` | No | false | Default open state (uncontrolled mode) |
| `modal` | `boolean` | No | - | Whether the popover is modal (blocks interaction with other elements) |
| `children` | `ReactNode` | Yes | - | Children |
