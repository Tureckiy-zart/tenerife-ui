## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `name` | `string` | Yes | - | Venue name (pre-localized string, required) |
| `description` | `string` | No | - | Venue description (pre-localized string, optional) |
| `location` | `string` | No | - | Location display string (pre-formatted address, optional) |
| `capacity` | `string` | No | - | Capacity display string (pre-formatted, optional) |
| `imageUrl` | `string` | No | - | Image URL (optional) |
| `href` | `string` | No | - | Link URL for venue details page (optional) |
| `eventsCount` | `number` | No | 0 | Number of associated events (optional) |
| `featured` | `boolean` | No | false | Whether this is a featured venue |
| `showImage` | `boolean` | No | true | Show image section |
| `eventsLabel` | `string` | Yes | - | Label for events count (required) |
| `popularBadgeText` | `string` | No | - | Badge text for popular venues (optional) |
| `capacityLabel` | `string` | Yes | - | Label for capacity (required) |
| `className` | `string` | No | - | Additional CSS classes |
| `animation` | `ComponentAnimationConfig` | No | - | Animation configuration for entrance and hover animations |
