## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `isOpen` | `boolean` | Yes | - | Whether panel is open |
| `onClose` | `() => void` | Yes | - | Callback when panel should close |
| `groupBy` | `GroupByFunction` | No | groupByDate | Function to group notifications (by date, type, or custom) |
| `width` | `"sm" | "md" | "lg"` | No | md | Panel width variant |
| `autoCollapse` | `boolean` | No | true | Auto-collapse older notifications |
| `returnFocusRef` | `React.RefObject<HTMLElement>` | No | - | Element to return focus to when panel closes |
