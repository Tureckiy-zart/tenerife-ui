## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `items` | `BreadcrumbItem[]` | Yes | - | Array of breadcrumb items |
| `separator` | `ReactNode` | No | ChevronRight icon | Custom separator component or string
    @default ChevronRight icon |
| `ariaLabel` | `string` | No | Breadcrumb | ARIA label for the navigation element
    @default "Breadcrumb" |
| `isLast` | `boolean` | No | - | Whether this is the last item (current page) |
| `disabled` | `boolean` | No | - | Whether this item is disabled |
| `children` | `ReactNode` | No | - | Custom separator content |
